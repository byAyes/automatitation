<name>Task 2: Create job history management module</name>
<files>src/lib/automation/job-history.ts</files>
<action>
Create `src/lib/automation/job-history.ts` with three exported functions:

1. `filterNewJobs(jobs: Job[])`: 
   - Accepts array of Job objects
   - Queries database for existing job URLs
   - Returns only jobs where URL doesn't exist OR emailedAt is null
   - This filters out duplicates and already-emailed jobs

2. `markJobsAsEmailed(jobIds: string[])`:
   - Accepts array of job IDs
   - Updates all matching jobs with emailedAt = new Date()
   - Returns count of updated jobs

3. `cleanupOldJobs(retentionMonths: number = 3)`:
   - Accepts retention period in months (default 3)
   - Deletes jobs where scrapedAt is older than retention period
   - Returns count of deleted jobs

Use Prisma client from `@/lib/prisma` (existing pattern).
Handle errors gracefully - log but don't throw for non-critical failures.
</action>
