<name>Task 1: Create scheduler orchestration module</name>
<files>src/automation/scheduler.ts, src/automation/orchestrator.ts</files>
<action>
Create two modules:

1. `src/automation/orchestrator.ts` - Pipeline executor:
   - `executePipeline()` function that:
     a. Runs all 3 scrapers (LinkedIn, Indeed, Glassdoor) via existing scraper index
     b. Calls job matching API or scorer directly
     c. Filters jobs through filterNewJobs() from job-history.ts
     d. If jobs exist: formats digest via formatJobDigest(), sends via sendEmail()
     e. Marks sent jobs as emailed via markJobsAsEmailed()
     f. Runs cleanupOldJobs() for retention
   - Returns summary: { scraped: number, matched: number, sent: number, cleaned: number }
   - Handles partial failures - continues if one scraper fails

2. `src/automation/scheduler.ts` - Entry point:
   - `runAutomation()` function that:
     a. Initializes logger
     b. Calls executePipeline()
     c. Logs results
     d. Handles top-level errors
   - Main guard: if require.main === module, run runAutomation()

Use existing scraper index, email modules, and job-history functions.
Log each step with logger.info() and logger.success().
</action>
